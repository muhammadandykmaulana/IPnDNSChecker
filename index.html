Latar Belakang & Tujuan

Latar Belakang

Meningkatnya ancaman kejahatan siber (malware, phishing, spam) menuntut adanya alat peringatan dini yang sederhana untuk memvalidasi keamanan suatu IP atau Domain secara real-time.

Tujuan Proyek

  • Tugas UTS: Integrasi API publik menggunakan Flask.
  • HTTP Request: Pengambilan data dinamis via library requests.
  • Data Rendering: Parsing JSON kompleks ke HTML via Jinja2.
  • Deployment: Implementasi Reverse Proxy via Cloudflare Tunnel.

Fitur Utama Aplikasi

Mengecek apakah sebuah target aman diakses berdasarkan data dari sumber global.

Pencarian IP Address

Terintegrasi dengan AbuseIPDB API.

Mendeteksi aktivitas berbahaya (DDoS, spam, malware). Menampilkan Skor Risiko dan tabel riwayat laporan detail.

Pencarian Nama Domain

Terintegrasi dengan VirusTotal API v3.

Mengecek status website dari puluhan vendor antivirus global. Mencegah akses ke web phishing atau distribusi malware.

Tech Stack & Infrastruktur

Back-End Python 3, Flask, Requests
Front-End HTML5, CSS3, JS (Vanilla)
Data Sources AbuseIPDB & VirusTotal API

Arsitektur Deployment

Dikembangkan di GitHub Codespaces (tertutup). Menggunakan Cloudflare Tunnel untuk expose port 5000 ke internet publik.

Instalasi & Penggunaan Lokal

# 1. Clone & Masuk direktori
git clone https://github.com/muhammadandykmaulana/IPnDNSChecker.git
cd IPnDNSChecker
# 2. Install dependensi
pip install Flask requests
# 3. Konfigurasi (app.py)
ABUSEIPDB_API_KEY = 'API_KEY_ANDA'
VIRUSTOTAL_API_KEY = 'API_KEY_ANDA'
# 4. Jalankan aplikasi lokal (Akses localhost:5000)
python app.py

Deployment via Cloudflare Tunnel (1/2)

Mengekspos local server (Codespace) ke internet publik dengan aman.

1. Install Cloudflared

# Contoh untuk OS Debian/Ubuntu curl -L --output cloudflared.deb https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64.deb sudo dpkg -i cloudflared.deb

2. Login & Otentikasi

# Buka tautan di browser untuk otentikasi akun Cloudflare Anda cloudflared tunnel login

Deployment via Cloudflare Tunnel (2/2)

3. Setup Custom Domain (Permanen)

# Buat tunnel baru
cloudflared tunnel create namatunnel
# Hubungkan tunnel ke domain/subdomain
cloudflared tunnel route dns namatunnel ipndns.domain.com

4. Cara Instan (Quick Way)

Jika hanya ingin mengekspos port sementara tanpa setup domain di dashboard, cukup gunakan perintah ini saat Flask berjalan:

cloudflared tunnel --url http://localhost:5000

Kesimpulan

Flask efisien: Terbukti sangat handal untuk merutekan permintaan pengguna dan menjembataninya dengan API eksternal.

Integrasi Berhasil: Mampu menarik data real-time dari sumber terpercaya (AbuseIPDB & VirusTotal).

Rendering Data Kompleks: Pemisahan antarmuka (HTML/CSS/JS) dengan Jinja2 mempermudah iterasi (looping) JSON kompleks menjadi tabel yang ramah bagi pengguna awam.
