Latar Belakang

  • Ancaman kejahatan siber terus meningkat.
  • IP dan Domain sering menjadi pintu masuk serangan (malware, phishing, spam).
  • Kebutuhan akan alat verifikasi keamanan jaringan yang mudah diakses.
  • Solusi: Membangun aplikasi berbasis web "Cyber Threat Analyzer" menggunakan Flask Python untuk analisis reputasi IP/Domain secara real-time.

Tujuan Proyek

  1. Mengimplementasikan penggunaan framework Flask untuk layanan back-end dinamis.
  2. Mempraktikkan pengambilan dan pengolahan data JSON dari API publik menggunakan library requests.
  3. Menampilkan data respons API secara interaktif menggunakan metode looping (perulangan) pada template HTML Jinja2.

Teknologi yang Digunakan

  • Back-End: Python 3 + Framework Flask
  • Front-End: HTML5, CSS3, Jinja2 Templating Engine
  • Integrasi Pihak Ketiga: Modul requests (HTTP GET)
  • Deployment & Infrastruktur: GitHub Codespaces

Integrasi API Publik

Nama API Kegunaan Format
AbuseIPDB API v2 Mengambil skor kepercayaan, detail ISP, dan riwayat laporan aktivitas mencurigakan dari sebuah alamat IP. JSON
VirusTotal API v3 Mengambil hasil pemindaian dari puluhan mesin antivirus global (Kaspersky, BitDefender, dll) terhadap sebuah URL/Domain. JSON

Struktur Direktori

Mengadopsi standar pemisahan logika dan tampilan (MVC-style):

Cyber-Threat-Analyzer/
├── app.py                 # File utama routing Flask & logika API
├── requirements.txt       # Daftar library (dependencies)
├── .env                   # File tersembunyi untuk API Key
└── templates/             # Folder wajib untuk tampilan HTML
    ├── index.html         # Form input halaman utama
    └── result.html        # Halaman tabel hasil analisis

Panduan Instalasi Lokal

  1. Clone Repo:
    git clone https://github.com/muhammadandykmaulana/IPnDNSChecker.git
  2. Masuk Folder & Buat Virtual Environment:
    cd IPnDNSChecker && python -m venv venv
  3. Aktivasi Venv & Install Library:
    source venv/bin/activate (Linux/Mac) atau venv\Scripts\activate (Win)
    pip install -r requirements.txt
  4. Set API Key: Masukkan key AbuseIPDB & VirusTotal ke dalam file .env.
  5. Jalankan:
    python app.py lalu buka http://127.0.0.1:5000 di browser.

Alur Kerja Sistem

  1. Input Pengguna: Memilih tipe (IP/Domain) dan memasukkan target di halaman utama.
  2. Pemrosesan Back-End: Flask menerima POST request, menyusun Header (API Key), dan menembak endpoint API yang sesuai.
  3. Pengolahan Data: Parsing data mentah JSON, menyaring array dan dictionary yang dibutuhkan.
  4. Rendering: Data dikirim ke result.html. Looping Jinja2 menyusun laporan menjadi tabel data.
